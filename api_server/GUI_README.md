# GUI для OCT Generators API

## Что добавлено

Создан современный веб-интерфейс для работы с Virtual Scanner API.

### Файлы GUI:

- `static/index.html` - главная страница интерфейса
- `static/style.css` - стили (современный дизайн с градиентами)
- `static/app.js` - JavaScript логика (загрузка файлов, обработка, отображение результатов)

## Особенности интерфейса

**Современный дизайн** - градиенты, плавные анимации, адаптивная верстка
**Drag & Drop** - можно перетаскивать файлы прямо в браузер
**Визуальная обратная связь** - статусы обработки, спиннеры, цветовая индикация
**Просмотр результатов** - автоматическое отображение сгенерированных OCT-сканов
**Скачивание** - кнопки для скачивания результатов

## Как использовать

### 1. Запустить сервер

```bash
python3 api_server.py --host 0.0.0.0 --port 8000
```

### 2. Открыть в браузере

```
http://your-server-ip:8000
```

### 3. Использовать интерфейс

1. Нажать "Choose File" или перетащить .dat файл
2. Нажать "Process File"
3. Дождаться обработки (может занять минуту)
4. Просмотреть результаты (grayscale и hot colormap)
5. Скачать изображения

## Важно: API остаётся доступным

**Все API endpoints работают как раньше:**
- `GET /health` - проверка работоспособности
- `POST /api/v1/scanner/process` - обработка файла
- `GET /api/v1/scanner/download/{session_id}/{image_type}` - скачивание результата
- `GET /docs` - автоматическая документация Swagger UI

**GUI - это дополнительный слой**, который использует те же API endpoints

**Ничего не сломано** - все существующие скрипты и API продолжают работать

## Технические детали

- GUI использует чистый JavaScript (без фреймворков)
- Все запросы идут через существующие API endpoints
- Статические файлы обслуживаются через FastAPI StaticFiles
- Интерфейс адаптивный (работает на мобильных устройствах)

## Скриншоты функционала

Интерфейс включает:
- Красивый заголовок с градиентом
- Зону загрузки файлов с drag & drop
- Индикаторы статуса (обработка, успех, ошибка)
- Карточки с результатами (grayscale и hot colormap)
- Кнопки скачивания

## Совместимость

- Работает во всех современных браузерах
- Поддерживает загрузку файлов через выбор или drag & drop
- Адаптивный дизайн для разных размеров экрана
