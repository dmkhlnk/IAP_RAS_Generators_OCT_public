# Structural Holes Generator

## Описание

Экспериментальная ветка генератора синтетических OCT сканов, создающая изображения с характерными структурными особенностями.

## Визуальные характеристики

Генерируемые сканы визуально напоминают:

- **Ткани кишечника** - характерная структура с полостями и перегородками
- **Ткани с рваными ранами** - нерегулярные разрывы и структурные дефекты

## Технические детали

- **Генератор**: `generator_v1.py` - модифицированная версия основного генератора
- **Виртуальный сканер**: `virtual_scanner.py` - симуляция OCT физики
- **Выходные данные**: 
  - Scatterers (.dat файлы) - координаты рассеивателей
  - OCT изображения (.png) - grayscale и hot colormap версии

## Использование

```bash
python3 generator_v1.py \
  --output_dir ./output \
  --num_variations 1 \
  --project_dir /path/to/project
```

## Результаты

Пример генерации:
- **Scatterers**: 213,927 рассеивателей
- **Изображения**: 
  - `patient01_slice00_var0_scatterers_grayscale.png` (1.4 MB)
  - `patient01_slice00_var0_scatterers_hot.png` (903 KB)
- **Данные**: `patient01_slice00_var0_scatterers.dat` (12 MB)

## Примечания

Это экспериментальная ветка, созданная для исследования альтернативных визуальных паттернов в синтетических OCT сканах. Результаты могут отличаться от основного генератора по структуре и морфологии тканей.

## Файлы

- `generator_v1.py` - основной генератор
- `virtual_scanner.py` - виртуальный OCT сканер
- `output/` - директория с результатами генерации
